# ⚠️ 易错点与避坑指南

> 基于学霸笔记 **3700+ 考点** 中提取的常考陷阱、易混淆概念和记忆口诀。
> 🔴 红 = 最容易错 | 🟡 黄 = 中等难度 | 🟢 绿 = 注意即可

---

## 一、法律类（易错最多）

### 🔴 宪法易混点

| 易混概念 | 辨析 | 记忆口诀/技巧 |
|----------|------|-------------|
| 修改宪法 vs 解释宪法 | 修改=全国人大，解释=全国人大常委会 | "人大改，常委解" |
| 制宪主体 vs 制宪机关 | 主体=人民，机关=第一届全国人大第一次全体会议 | — |
| 违宪审查 | 全国人大及其常委会共同监督宪法实施 | — |
| 村长 vs 居委会主任 | 村委会/居委会任期均为**5年** | 2018年修宪后统一为5年 |
| 人身自由 vs 人格尊严 | 人身自由是**最基本的权利** | 注意题目问法 |

### 🔴 行政法易混点

| 易混概念 | 辨析 |
|----------|------|
| **抽象行政行为 vs 具体行政行为** | 抽象=制定规则（可复议不可诉），具体=针对个人（可复议可诉）|
| **行政强制措施 vs 行政强制执行** | 措施=暂时性限制（查封/扣押/冻结），执行=最终处理（划拨/拍卖）|
| **派出机关 vs 派出机构** | 派出机关（行署/区公所/街道办）=职权性主体，派出机构（派出所/工商所）=授权性主体 |
| **行政处罚听证 vs 行政许可听证** | 处罚听证是依申请，许可听证可依职权或申请 |
| **复议前置的情形** | 自然资源所有权、纳税争议等必须先复议再起诉 |

### 🔴 刑事诉讼法 vs 民事诉讼法

| 维度 | 刑事诉讼法 | 民事诉讼法 |
|------|-----------|-----------|
| 起诉方 | 检察院（公诉）或自诉人 | 原告（公民/法人） |
| 被告 | 犯罪嫌疑人/被告人 | 被告（公民/法人） |
| 举证责任 | 控方举证（公诉） | **谁主张谁举证**（一般原则） |
| 证明标准 | 排除合理怀疑 | 优势证据 |

### 🟡 刑法易混点

| 易混概念 | 辨析 |
|----------|------|
| **正当防卫 vs 紧急避险** | 前者对"人"，后者对"第三人"的合法权益 |
| **一般防卫 vs 特殊防卫** | 特殊防卫针对严重危及人身安全的暴力犯罪（杀人/抢劫/强奸/绑架等），造成伤亡不负责 |
| **犯罪未遂 vs 犯罪中止** | 未遂=意志以外原因没得逞，中止=**自动**放弃或防止结果发生 |
| **主刑 5 种顺序** | 管制→拘役→有期徒刑→无期徒刑→死刑（由轻到重） |
| **累犯 vs 再犯** | 累犯=前罪有期徒刑以上+5年内再犯应当有期徒刑以上+都是故意+年满18 |
| **死刑缓期执行** | 死缓不是独立刑种，是死刑的一种执行制度 |
| **不得假释的情形** | 累犯+暴力性犯罪被判处10年以上有期徒刑/无期徒刑 |

### 🟡 民法易混点

| 易混概念 | 辨析 |
|----------|------|
| **宣告失踪 vs 宣告死亡** | 失踪2年(公告3月)，死亡4年(公告1年)/意外2年 |
| **普通诉讼时效(3年) vs 最长诉讼时效(20年)** | 普通从"知道或应当知道"起算，最长从"权利受损害之日"起算 |
| **物权 vs 债权** | 物权=绝对权/对世权，债权=相对权/对人权 |
| **过错责任 vs 无过错责任** | 一般侵权=过错，产品责任/高度危险=无过错 |
| **遗嘱效力顺序** | 公证遗嘱不再优先 → **以最后的遗嘱为准** |

### 🟡 劳动法易混点

| 易混数字 | 内容 |
|----------|------|
| **150% vs 200% vs 300%** | 平日加班150%，休息日不补休200%，法定节假日300% |
| **试用期上限** | 1年合同≤1月，3年合同≤2月，无固定≤6月 |
| **未签劳动合同** | 超过1月不满1年→付双倍工资(**最多11个月**) |
| **最低工资** | 试用期工资≥合同约定80%且≥当地最低工资标准 |

## 二、政治类

### 🔴 马原易混

| 易混概念 | 辨析 |
|----------|------|
| **古代朴素唯物 vs 近代形而上学唯物 vs 辩证唯物** | 古代=具体物质（水/火/气），近代=原子，辩证唯物=客观实在性 |
| **主观唯心 vs 客观唯心** | 主观=人的精神（"我思故我在"），客观=绝对精神/理念（"理在事先"） |
| **"扬弃"** | 既克服又保留，不是简单的否定 |
| **相对静止** | 是运动的特殊状态，不是不动 |
| **意识 vs 物质** | 意识是客观存在的主观映像，是人脑的机能 |

### 🔴 党史重要会议排序

```
一大(建党)→二大(反帝反封建)→三大(国共合作)→八七会议(武装斗争)
→古田会议(政治建军)→遵义会议(独立自主)→七大(毛思想)→
七届二中(农村→城市)→八大→十一届三中全会(改革开放)
→十二大(中国特色社会主义)→十三大(初级阶段)→
十五大(邓小平理论)→十六大(三个代表)→十七大(科学发展观)→
十九大(习近平新时代中国特色社会主义思想)
```

### 🟡 中特体系易混

| 易混 | 辨析 |
|------|------|
| 邓小平理论 vs 习近平思想 | 前者1997年十五大确立，后者2017年十九大确立 |
| 科学发展观 vs 新发展理念 | 前者强调"全面协调可持续"，后者"创新协调绿色开放共享" |

## 三、公文类

### 🔴 最容易混淆的文种

| 易混组 | 关键区别 |
|--------|----------|
| **公告 vs 通告** | 公告→**国内外**宣布重大事项；通告→社会各有关方面 |
| **请示 vs 报告** | 请示→**需要批复**，一文一事；报告→**不需要批复**，不得夹带请示 |
| **通知 vs 通报** | 通知→布置工作/告知事项；通报→**表彰批评**/传达情况 |
| **决定 vs 决议** | 决定→机关发文；决议→**会议**讨论通过 |

### 🟡 格式易错

| 易错点 | 正确做法 |
|--------|----------|
| 发文机关标志 | 红色**小标宋体**字 |
| 命令(令) | 发文机关标志+令+第×号 |
| 成文日期 | 领导人**签发**日期 |
| 页码 | 4号半角宋体阿拉伯数字 |
| 用纸 | A4型纸 |

## 四、经济类

### 🟡 易混概念

| 概念 | 说明 |
|------|------|
| **替代品(正比) vs 互补品(反比)** | 替代品涨价→本商品需求增；互补品涨价→本商品需求减 |
| **买方市场 vs 卖方市场** | 买方=供过于求(价降)，卖方=供不应求(价涨) |
| **财政政策(政府) vs 货币政策(央行)** | 经济过热→增税/提利率；经济萧条→减税/降利率 |
| **GDP vs GNP** | GDP=地域概念，GNP=国民概念 |

### 🟢 常考数字

| 恩格尔系数 | 含义 |
|-----------|------|
| < 30% | 最富裕 |
| 30%-40% | 富裕 |
| 40%-50% | 小康 |
| 50%-60% | 温饱 |
| > 60% | 贫困 |

## 五、管理类

### 🟡 易混概念

| 易混 | 辨析 |
|------|------|
| 激励因素 vs 保健因素 | 激励=工作本身(成就/认可)，保健=工作环境(薪资/条件) |
| 马斯洛五层次顺序 | 🔴 易考排序题：生理→安全→社交→尊重→自我实现 |
| 管理幅度 vs 管理层次 | 规模一定时**反比** |
| 晕轮效应 vs 首因效应 | 晕轮=以点带面，首因=第一印象 |
