# 习近平新时代中国特色社会主义思想

> 马克思主义中国化的最新成果，2017年党的十九大确立为党的指导思想，2018年写入宪法。
> 学霸笔记考点数（含中特体系）：**409个**

---

## 历史地位

- 马克思主义中国化的**最新成果**
- 党和人民实践经验和集体智慧的结晶
- 中国特色社会主义理论体系的重要组成部分
- 全党全国人民为实现中华民族伟大复兴而奋斗的**行动指南**
- 2017年**十九大**写入党章
- 2018年**宪法修正案**写入宪法

---

## 核心内容

### "十个明确"
（待从教材补充详细表述）

1. 明确中国特色社会主义最本质的特征是**中国共产党领导**
2. 明确坚持和发展中国特色社会主义，总任务是实现社会主义现代化和中华民族伟大复兴
3. 明确新时代我国社会主要矛盾是人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾
4. 明确中国特色社会主义事业总体布局是"五位一体"、战略布局是"四个全面"
5. 明确全面深化改革总目标是完善和发展中国特色社会主义制度、推进国家治理体系和治理能力现代化
6. 明确全面推进依法治国总目标是建设中国特色社会主义法治体系、建设社会主义法治国家
7. 明确党在新时代的强军目标是建设一支听党指挥、能打胜仗、作风优良的人民军队
8. 明确中国特色大国外交要推动构建新型国际关系，推动构建人类命运共同体
9. 明确全面从严治党的战略方针，提出新时代党的建设总要求
10. 明确坚持和完善社会主义基本经济制度，使市场在资源配置中起决定性作用，更好发挥政府作用

### 中国特色社会主义理论体系发展历程

| 会议 | 年份 | 重要成果 |
|------|------|----------|
| **十一届三中全会** | 1978 | 工作重心转移到现代化建设，改革开放 |
| **十二大** | 1982 | 提出"建设有中国特色社会主义" |
| **十三大** | 1987 | 社会主义初级阶段理论，"一个中心、两个基本点" |
| **南方谈话** | 1992 | 邓小平理论走向成熟 |
| **十五大** | 1997 | 正式提出"邓小平理论"，确立为指导思想 |
| **十六大** | 2002 | "三个代表"重要思想 |
| **十七大** | 2007 | 科学发展观 |
| **十八大** | 2012 | 全面建成小康社会 |
| **十九大** | 2017 | **习近平新时代中国特色社会主义思想** |
