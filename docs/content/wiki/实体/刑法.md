# 中华人民共和国刑法

> 规定犯罪、刑事责任和刑罚的法律。
> 学霸笔记考点数：**86个**

---

## 犯罪构成

| 要件 | 内容 | 说明 |
|------|------|------|
| **犯罪主体** | 自然人和单位 | 刑事责任年龄：14-16（8种罪）→ 16+（全责） |
| **犯罪客体** | 刑法保护的社会关系 | 区别于犯罪对象 |
| **犯罪主观方面** | 故意、过失 | 故意（直接/间接）、过失（疏忽/过于自信） |
| **犯罪客观方面** | 行为、结果、因果关系 | 危害行为是核心 |

### 刑事责任年龄
- **不满14周岁**：不负刑事责任
- **14-16周岁**：犯**故意杀人、故意伤害致人重伤或死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投放危险物质罪**的，负刑事责任（8种罪）
- **16周岁以上**：完全刑事责任
- **已满14不满18**：从轻或减轻处罚
- **已满75周岁**：故意犯罪可从轻，过失犯罪应从轻

## 正当防卫与紧急避险

### 正当防卫
- 条件：针对**正在进行**的不法侵害
- 限度：不能明显超过必要限度
- ★ **特殊防卫（无限防卫权）**：针对严重危及人身安全的暴力犯罪（杀人、抢劫、强奸、绑架等），造成伤亡不负责任

### 紧急避险
- 条件：**不得已**损害另一较小合法权益
- 不适用于职务上、业务上有特定责任的人

## 犯罪形态

| 形态 | 定义 | 处罚 |
|------|------|------|
| **犯罪预备** | 准备工具、制造条件 | 可比照既遂从轻/减轻/免除 |
| **犯罪未遂** | 已着手，因意志外原因未得逞 | 可比照既遂从轻/减轻 |
| **犯罪中止** | 自动放弃或自动防止结果发生 | 无损害→免除，有损害→减轻 |
| **犯罪既遂** | 行为完全实现犯罪构成 | 按该罪处罚 |

### 共同犯罪
- **主犯**：组织、领导犯罪集团 → 按全部罪行处罚
- **从犯**：次要或辅助作用 → 从轻/减轻/免除
- **胁从犯**：被胁迫参加 → 减轻/免除
- **教唆犯**：教唆他人犯罪 → 按在共同犯罪中的作用处罚

## 刑罚种类

### 主刑（只能独立适用）
| 种类 | 期限 | 说明 |
|------|------|------|
| **管制** | 3个月-2年 | 不关押，社区矫正，同工同酬 |
| **拘役** | 1个月-6个月 | 就近关押，每月可回家1-2天 |
| **有期徒刑** | 6个月-15年 | 数罪并罚不超过25年 |
| **无期徒刑** | 终身 | 可减刑、假释 |
| **死刑** | — | ★ 报请最高院核准 |

### 附加刑（可独立或附加适用）
- **罚金**
- **剥夺政治权利**
- **没收财产**
- **驱逐出境**（适用于外国人）

### 死刑的特殊规定
- 犯罪时不满18周岁、审判时怀孕的妇女、审判时已满75周岁 → **不适用死刑**
- 死刑立即执行 → 报**最高人民法院**核准
- 死缓：2年期满可减为无期或有期25年

## 常考罪名

| 类型 | 常见罪名 |
|------|----------|
| 危害公共安全罪 | 放火、爆炸、交通肇事、危险驾驶 |
| 侵犯人身权利罪 | 故意杀人、故意伤害、强奸、绑架 |
| 侵犯财产罪 | 抢劫（最高可死刑）、盗窃、诈骗、抢夺 |
| 贪污贿赂罪 | 贪污、受贿、行贿、挪用公款 |
| 渎职罪 | 滥用职权、玩忽职守 |
