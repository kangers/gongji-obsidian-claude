# 监察法

> ★ 公基法律高频考点。大纲明确列出。2018年宪法修正案关联考点。
> 核心法规：《中华人民共和国监察法》（2018.3.20施行）

---

## 监察委员会概述

### 性质
- 各级监察委员会是行使国家**监察职能**的**专责机关**
- 是**政治机关**，不是行政机关、也不是司法机关
- 依法对所有行使公权力的**公职人员**进行监察

### 与"一府两院"的并列关系（★ 2018宪法修正案）
- 国家机构体系：人大 → 一府（政府） + 一委（**监察委**） + 两院（法院、检察院）
- 监察委由**人大**产生，对**人大**负责，受**人大**监督
- 在人大闭会期间，对**人大常委会**负责并报告工作

---

## 监察对象（★ 六类人员全覆盖）

| 序号 | 对象 | 举例 |
|:--:|------|------|
| 1 | 中国共产党机关、人大机关、行政机关、政协机关、监察机关、审判机关、检察机关、民主党派和工商联机关的**公务员**，及参照公务员法管理的人员 | 各级公务员 |
| 2 | 法律法规授权或受国家机关依法委托管理公共事务的组织中**从事公务**的人员 | 行业协会管理人员 |
| 3 | **国有企业**管理人员 | 董事长、总经理等 |
| 4 | **公办**教育、科研、文化、医疗卫生、体育等单位中**从事管理**的人员 | 公立学校校长、医院院长 |
| 5 | 基层群众性自治组织中**从事管理**的人员 | 村委会主任、居委会主任 |
| 6 | **其他**依法履行公职的人员 | 人大代表等 |

> ★ **私营企业主**不属于监察对象！考察是否行使"公权力"

---

## 监察机关的职责（三项职责）

| 职责 | 内容 |
|------|------|
| **监督** | ★ 首要职责。对公职人员开展廉政教育，对其依法履职、秉公用权、廉洁从政从业及道德操守进行监督检查 |
| **调查** | 对涉嫌贪污贿赂、滥用职权、玩忽职守、权力寻租、利益输送、徇私舞弊以及浪费国家资财等**职务违法和职务犯罪**进行调查 |
| **处置** | 对违法公职人员依法作出政务处分决定；对履行职责不力、失职失责领导人员进行问责；对涉嫌职务犯罪的移送检察院 |

---

## 监察权限（12项调查措施）

| 类型 | 措施 | 说明 |
|------|------|------|
| 谈话/讯问 | 谈话、讯问、询问 | 对涉嫌职务违法的被调查人 |
| 查询/冻结 | 查询、冻结 | 冻结财产 |
| 搜查/调取 | 搜查、调取、查封、扣押（暂扣） | 搜查住所、扣押证件 |
| 勘验/鉴定 | 勘验检查、鉴定 | 现场勘查 |
| 技术调查 | **技术调查** | 通信监控等 |
| 限制出境 | **限制出境** | 防止逃匿 |
| **留置** | ★ **留置** | ★ 取代"双规"的法定措施 |

---

## 留置（★ 最高频考点）

### 留置的条件
1. 涉及案情**重大、复杂的**
2. 可能**逃跑、自杀**的
3. 可能**串供**或伪造、隐匿、毁灭证据的
4. 可能有**其他妨碍调查**行为的

### 留置的程序
| 项目 | 内容 |
|------|------|
| 审批权限 | 由监察机关**领导人员集体研究决定**（设区的市级以下需报上一级批准） |
| 留置期限 | 一般不超过**3个月** |
| 延长期限 | 可延长一次，不超过**3个月**（即最长6个月） |
| 通知义务 | 留置后**24小时**内通知单位和家属（有碍调查除外） |
| 刑期折抵 | 留置一日折抵管制**二日**，折抵拘役/有期徒刑**一日** |

---

## 处置方式

| 处置方式 | 适用条件 |
|------|------|
| **谈话提醒/批评教育/责令检查/诫勉** | 有职务违法行为但情节较轻 |
| **政务处分**（警告/记过/记大过/降级/撤职/开除） | 违法 |
| **问责**（通报/诫勉/组织调整/处分） | 领导人员失职失责 |
| **移送起诉** | 涉嫌职务犯罪 |
| **提出监察建议** | 对监察对象所在单位廉政建设和履职问题 |

---

## 监察机关与司法机关的关系

| 衔接环节 | 说明 |
|------|------|
| **证据衔接** | 监察机关依法收集的证据材料，在刑事诉讼中**可以作为证据使用** |
| **移送审查起诉** | 监察机关调查终结 → 移送检察院 → 检察院审查起诉 |
| **补充调查** | 检察院认为需要补充核实的 → 退回监察机关补充调查（必要时可自行补充侦查） |

---

## 对监察机关的监督

- 人大监督：各级监察委员会向本级**人大常委会**作专项工作报告
- 民主监督/社会监督/舆论监督
- 内部监督：设立内部专门监督机构
- 监察人员违法：被调查人及其近亲属有权**申诉**

## 常考辨析

| 易混项 | 正确 |
|------|------|
| 监察委的性质 | **政治机关**（不是行政机关/司法机关） |
| 监察委的首要职责 | **监督**（不是调查） |
| 监察对象 | 所有行使公权力的**公职人员** |
| 私营企业主 | ❌ 不属于监察对象 |
| 留置最长期限 | **6个月**（3+3） |
| 留置通知家属 | **24小时内** |
| 留置一日折抵刑期 | 管制二日，拘役/有期一日 |
| "双规"的替代 | **留置** |
