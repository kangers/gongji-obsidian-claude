# 公务员法

> ★ 公基法律高频考点。大纲明确列出。
> 核心法规：《中华人民共和国公务员法》（2018修订，2019.6.1施行）

---

## 公务员的定义与范围

### 公务员定义
- 依法履行**公职**
- 纳入国家**行政编制**
- 由国家财政负担**工资福利**的工作人员

> **三条件缺一不可**："公职 + 行政编制 + 国家财政"

### 范围（七类机关工作人员）
| 序号 | 机关 |
|:--:|------|
| 1 | 中国共产党各级机关 |
| 2 | 各级人民代表大会及其常务委员会机关 |
| 3 | 各级行政机关 |
| 4 | 各级政治协商会议机关 |
| 5 | 各级监察委员会 |
| 6 | 各级审判机关（法院） |
| 7 | 各级检察机关（检察院） |

### ★ 不属于公务员范围（常考）
- 人事关系在**事业单位**的人大代表 ❌
- 国有企业管理人员 ❌
- 村委会/居委会成员 ❌
- 工勤人员 ❌

## 公务员职位分类

### 三大职位类别
| 类别 | 范围 |
|------|------|
| **综合管理类** | 机关中履行规划、咨询、决策、组织、指挥、协调、监督等职责 |
| **专业技术类** | 专门从事专业技术工作 |
| **行政执法类** | 在市场监管、生态环境保护、文化市场等执法队伍中 |

### 职务与职级并行制度
- **领导职务序列**（10级）：国家级正职 → 国家级副职 → 省部级正职 → …… → 乡科级副职
- **职级序列**（12级）：在厅局级以下设置（一级巡视员 → 二级巡视员 → 一级调研员 → …… → 二级科员）

> ★ 2018年修订的最大亮点之一：推行**职务与职级并行**，打破晋升"天花板"

## 公务员的录用与考核

### 录用条件
- 年满**18周岁**
- 具有中华人民共和国国籍
- 具有正常履职的身体条件
- 具有符合职位要求的文化程度和工作能力

### 不得录用情形（常考多选）
1. 因**犯罪**受过刑事处罚的
2. 被开除**中国共产党党籍**的（★ 2018新增）
3. 被开除**公职**的
4. 被依法列为失信联合惩戒对象的（★ 2018新增）

### 考核
| 项目 | 内容 |
|------|------|
| 考核形式 | **平时考核 + 专项考核 + 定期考核** |
| 定期考核结果 | 优秀、称职、基本称职、**不称职** |
| ★ 重点考核 | **政治素质**和**工作实绩** |

> 连续两年年度考核不称职 → 予以辞退

## 公务员交流

### 交流方式（★ 2018修订后）

| 方式 | 说明 |
|------|------|
| **调任** | 国有企业/高校/科研院所等调入机关担任领导职务或四级调研员以上 |
| **转任** | 公务员在不同职位之间的交流 |

> ★ **挂职锻炼不再属于公务员交流方式！** （2018年修订删除）

---

## 回避制度

| 回避类型 | 内容 |
|------|------|
| **任职回避** | 夫妻/直系血亲/三代以内旁系血亲/近姻亲 → 不得在同一机关担任双方直接隶属于同一领导人员的职务或者有直接上下级领导关系的职务 |
| **地域回避** | 乡级/县级主要领导职务，不得在**本人成长地**任职 |
| **公务回避** | 涉及本人或亲属利害关系的，应回避 |

### ★ 公务员退休后从业限制（高频）
- 领导成员离职**3年内**，其他公务员离职**2年内**
- 不得在原工作业务**直接相关**的营利性组织任职
- 不得从事与原工作业务**直接相关**的营利性活动

---

## 辞职与辞退

### 不得辞去公职的情形
1. 未满国家规定的**最低服务年限**
2. 在涉密岗位任职，不满脱密期限
3. 重要公务尚未处理完毕，须本人继续处理
4. 正在接受审计/纪律审查/监察调查

### 应予以辞退的情形
- 连续两年年度考核**不称职**
- 不胜任现职工作，又不接受其他安排
- 因所在机关调整/撤销，拒绝合理安排
- 旷工连续超过**15天**或一年内累计超过**30天**

### 不得辞退的情形
1. 因公致残，被确认丧失或部分丧失工作能力
2. 患病或负伤，在**规定医疗期内**
3. 女性公务员在**孕期/产假/哺乳期内**

## 处分

### 处分种类（共6种）

| 序号 | 处分 | 处分期间 | 说明 |
|:--:|------|:--:|------|
| 1 | **警告** | 6个月 | 最轻 |
| 2 | **记过** | 12个月 | |
| 3 | **记大过** | 18个月 | |
| 4 | **降级** | 24个月 | |
| 5 | **撤职** | 24个月 | 同时降低级别 |
| 6 | **开除** | — | ★ 永久，最重处分 |

> **受处分期间不得晋升职务、职级和级别**（警告除外，警告期间仍可晋升级别）

## 权利救济

| 救济途径 | 适用情形 |
|------|------|
| **复核** | 对涉及本人的人事处理不服 |
| **申诉** | 对复核结果不服 → 向上级机关或同级公务员主管部门申诉 |
| **控告** | 认为机关及其领导人员侵犯其合法权益 |

## 高频数字速查

| 考点 | 数字 |
|------|:--:|
| 报考年龄下限 | **18岁** |
| 领导成员离职禁止期 | **3年** |
| 其他公务员离职禁止期 | **2年** |
| 年度考核"不称职"连续 | **2年**（辞退） |
| 旷工连续/累计 | **15天/30天** |
| 警告处分期 | **6个月** |
| 记过 | **12个月** |
| 记大过 | **18个月** |
| 降级/撤职 | **24个月** |
