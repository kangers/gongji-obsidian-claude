# 管理

> 考试权重：**15%-20%**
> 学霸笔记考点数：**92个**

---

## 管理学基本原理

### 管理概述
- 管理的核心是**决策**
- 管理的两重性：**自然属性**和**社会属性**
- 管理的四个基本职能：**计划、组织、领导、控制**
- 计划是管理的**首要职能**
- 人力资源在组织各项资源中处于**核心地位**

### 管理者
- 管理者的信息传递角色：监督人、传播者、发言人
- 管理者的决策制定角色：企业家、资源分配者、冲突管理者、谈判者
- 管理者应具备的技能：技术技能、人际技能、概念技能
- 高层管理者更需要**概念技能**，基层管理者更需要**技术技能**

### 管理理论演进
- **泰勒** — "科学管理之父"，根本目的是谋求最高劳动生产率
- **法约尔** — 一般管理理论，14条管理原则
- **韦伯** — 官僚组织理论（科层制）
- **梅奥** — 人际关系理论（霍桑实验）

---

## 决策与组织

### 决策类型
- **战略决策** — 长期性、方向性、全局性
- **战术决策** — 中期、局部
- **业务决策** — 短期、具体
- **程序化决策** — 常规、重复
- **非程序化决策** — 新问题、不确定
- **风险型决策** — 知道自然状态的概率

### 组织设计
- 管理层次 vs 管理幅度：规模一定时，两者成**反比**
- 直线制、职能制、直线职能制、事业部制、矩阵制

---

## 激励与领导

### 马斯洛需求层次理论
- 五个层次：**生理** → **安全** → **社交** → **尊重** → **自我实现**
- 生理需要（最低层次）→ 自我实现需要（最高层次）
- 生理、安全 = 较低级需要（物质需求）
- 社交、尊重、自我实现 = 较高级需要（精神需求）

### 双因素理论（赫茨伯格）
- **激励因素** — 与工作本身相关（成就、认可、发展）
- **保健因素** — 与工作环境相关（薪资、条件、人际关系）

### 管理心理学效应
- **晕轮效应** — 以点带面、以偏概全
- **首因效应** — 第一印象
- **近因效应** — 最近一次接触的影响
- **木桶原理** — 取决于最短的木板
- **破窗效应** — 破损的东西更易遭到破坏
- **青蛙效应** — 渐进变化的忽视

---

## 控制
- 控制工作的标准：能否**适时纠正偏差**
- **事前控制** — 控制最高境界，预防问题发生
- **事中控制** — 工作进行中的控制
- **事后控制** — 工作结束后的控制，"亡羊补牢"

## 行政管理

> 大纲明确列出的独立模块。考试占比：管理科目的 40%-50%

### 政府职能

#### 政府职能概述
- 政府职能是**行政管理的基本问题**，是公共行政的**本质表现**
- 政府职能体现了**公共行政活动的基本内容和方向**
- 政府职能是建立行政组织的**根本依据**

#### 政府基本职能（四大职能）

| 职能 | 内容 |
|------|------|
| **政治职能** | 军事保卫、外交、治安、民主政治建设 |
| **经济职能** | 宏观调控、市场监管、提供公共产品和服务 |
| **文化职能** | 科技、教育、文化、卫生、体育 |
| **社会职能** | 社会保障、环境保护、人口控制、公共服务 |

#### 政府职能转变
- 核心：**政企分开**（转变政府职能的关键）
- 方向：从**管理型**政府向**服务型**政府转变
- 简政放权、放管结合、优化服务（**"放管服"改革**）
- 判断行政管理体制改革成效的关键：能否**转变政府职能**

### 行政组织

#### 行政组织的构成要素
- 组织目标、机构设置、人员构成、权责体系、法规制度、物质因素

#### 行政组织类型

| 类型 | 说明 | 举例 |
|------|------|------|
| **领导机关** | 行政组织的中枢 | 各级政府 |
| **职能机关** | 分管专门事务 | 各部/厅/局 |
| **辅助机关** | 为行政首长或职能机关服务 | 办公室/研究室 |
| **直属机关** | 直接隶属于领导机关 | 统计局、环保局 |
| **派出机关** | 一级政府根据政务管理需要派出 | 行政公署、区公所、街道办事处 |

#### 行政组织结构

| 结构类型 | 特点 | 优点 | 缺点 |
|------|------|------|------|
| **直线式**（最基本） | 垂直领导，命令统一 | 结构简单、权责明确 | 缺乏专业化分工 |
| **职能式** | 按专业分工 | 专业化程度高 | 多头领导 |
| **直线职能式** | 直线+职能结合 | 兼具统一与专业 | 协调难度大 |
| **矩阵式** | 纵横结合，临时项目 | 灵活、资源利用率高 | 双重领导、稳定性差 |

### 行政领导

#### 行政领导的产生方式

| 方式 | 说明 | 举例 |
|------|------|------|
| **选任制** | 由选举产生 | 省长/市长/县长（人大选举） |
| **委任制** | 由上级机关直接任命 | 国务院各部部长 |
| **考任制** | 公开考试择优录用 | 公务员招录 |
| **聘任制** | 签订合同聘任 | 专业技术或辅助性职位 |

#### 行政领导体制

| 体制 | 特点 | 实行机关 |
|------|------|------|
| **首长负责制** | 行政首长拥有最后决定权 | 国务院（总理负责制）、各级政府 |
| **委员会制** | 集体讨论、少数服从多数 | 人大及其常委会 |
| **混合制** | 首长制与委员会制结合 | 部分事业单位 |

### 人事行政

- 公务员制度（详见 [公务员法](wiki/实体/公务员法)）
- 公务员录用、考核、奖惩、培训、交流、回避等

### 公共财政
- 国家预算是政府的基本财政收支计划
- 财政收入来源：税收（主体）、国有资产收益、国债、收费
- 财政支出：购买性支出 + 转移性支出
- 财政政策：扩张性（增加支出/减税）、紧缩性（减少支出/增税）

### 行政监督

| 监督类型 | 内容 | 举例 |
|------|------|------|
| **政党监督** | 执政党和参政党的监督 | 纪委监督、政协民主监督 |
| **国家权力机关监督** | ★ 行政监督体系的**核心** | 人大及其常委会 |
| **司法机关监督** | 法院和检察院的监督 | 行政诉讼、检察建议 |
| **社会监督** | 公民、社会组织、舆论 | 信访、举报、媒体 |
| **审计监督** | 专门经济监督 | 审计署/审计厅 |
| **行政机关内部监督** | 一般监督 + 专门监督 | 上级对下级、监察/审计 |

## 公共政策

> 大纲明确列出的独立模块。常考单选/多选。

### 公共政策概述
- 公共政策是公共权力机关为解决公共问题、达成公共目标、实现公共利益而制定的方案
- 本质：对**社会利益**的权威性分配

### 公共政策过程（★ 四阶段模型）

| 阶段 | 内容 |
|------|------|
| **政策制定** | ★ 核心环节。包括：问题界定 → 议程设立 → 方案规划 → 方案抉择 |
| **政策执行** | 将政策方案付诸实践。影响执行的因素：政策本身（合理/明确）、执行机构（组织/人员）、目标群体（配合/抵制）、政策环境（政治/经济/文化） |
| **政策评估** | 对政策效果进行判断。评估标准：**效率、效益、公平性、回应性** |
| **政策监控** | 全过程监督控制，保证政策目标实现 |

### 政策制定的模式（常考单选辨析）

| 模式 | 代表人物 | 核心观点 |
|------|------|------|
| **理性模式** | — | 完全信息 + 最优决策 |
| **渐进模式** | 林德布洛姆 | 政策是过去行为的**逐步修正**，不是全盘重新设计 |
| **混合扫描模式** | 埃齐奥尼 | 结合理性模式与渐进模式 |
| **垃圾桶模式** | 科恩 | 政策制定是问题、方案、参与者、选择机会的随机结合 |

### 影响公共政策执行的因素（多选常考）
1. 政策本身的**合理性**（目标是否清晰、措施是否可行）
2. 执行机构的**组织结构和执行能力**
3. 执行人对政策的**理解和认同度**
4. 政策目标群体的**行为状况和配合程度**
5. 政治、经济、文化等**外部环境**

### 政策执行偏差
- 象征性执行（做样子）
- 选择性执行（只执行有利的）
- 替换性执行（"上有政策、下有对策"）
- 附加性执行（随意扩大政策范围）
