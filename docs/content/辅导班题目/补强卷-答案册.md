## 补强③ 经济管理（12题）

| 题号 | 答案 | 简短解析 |
|------|------|----------|
| 1 | **B** | "看不见的手"=**亚当·斯密**；"看得见的手"(国家干预)=凯恩斯 |
| 2 | **C** | 四大市场机制中，**价格机制是核心**（基础配置机制） |
| 3 | **B** | 一种涨价→另一种需求增 = **替代品**（如猪肉涨→买鸡肉） |
| 4 | **B** | 商品二因素=**使用价值+价值**；交换价值是价值的表现形式 |
| 5 | **B** | 企业所得税基本税率=**25%**（不是20%！） |
| 6 | **C** | 管理的核心=**决策**（不是计划——计划是首要职能） |
| 7 | **A** | 四大职能顺序：**计划→组织→领导→控制**（前→锋→线） |
| 8 | **B** | 泰勒=**科学管理之父**；法约尔=一般管理理论之父 |
| 9 | **B** | 马斯洛最低=**生理需要**；最高=自我实现 |
| 10 | **C** | 激励因素=**工作本身**（成就感/认可）；保健因素=工作环境（工资/条件） |
| 11 | **B** | 幅度越大→层次越少：**反比关系** |
| 12 | **A** | 最高境界=**事前控制**（预防）>事中(过程)>事后(亡羊补牢) |

### 🔑 经济易混速记

| 概念 | 关键 |
|------|------|
| 看不见的手 | 亚当·斯密（自发的市场调节） |
| 看得见的手 | 凯恩斯（国家干预经济） |
| 价格机制 | 市场机制**核心** |
| **替代品**（正比） | 猪肉涨→买鸡肉多 |
| **互补品**（反比） | 汽油涨→买车少 |
| 商品二因素 | 使用价值+**价值**（不是交换价值！） |
| 企业所得税 | **25%** |
