```
民事诉讼法专项答案解析

一、单选是

1.【答案 ] C。解析: 民事诉讼法的属性有四个方面: (1 ) 从其在法律体系的地位
来分析，它居于基本法律的地位; (2 ) 从其调整的社会关系来分析，它属于独立的部门
法; (3 ) 从其所规定的内容来分析，它规定的主要是程序问题，属于程序法。 (4) 从
其适用的主体上看, 它规范国家机关与公民、法人、非法人团体在诉讼活动之间的关系，
属于公法。故本题答案为 C。

2.【答案 ] D。 解析: 教育局与商家进行的是平等主体之间的财产交易，因产品质量
问题协商无效提起的诉讼属于民事诉讼。故本题答案为D。

3.【答案 ] D。解析: 民事诉讼法的基本原则包括: 当事人诉讼权利平等原则、同等
原则和对等原则、法院调解自愿和合法的原则 、辩论原则、处分原则、诚信原则、检察
监督原则、支持起诉原则。故本题答案为D。
```

```
合法,证据的收集手段和收集程序合法。无因性不属于证据的特征。故本题答案为D。
```
