这个知识库采用了 Andrej Karpathy 提出的 [LLM Wiki](https://github.com/karpathy/llm-wiki) 模式——不是传统的上传文件然后等 AI 检索（RAG），而是让 AI **主动构建和维护一个持久化的 Wiki**。

```
raw/           → 原始来源（不可变）
wiki/          → LLM 维护的结构化知识库
LLM-WORKFLOW.md → Schema 配置（告诉 AI 怎么维护）
```

每次添加新资料，AI 不只索引它，而是：
- 提取关键信息
- 更新相关概念页面
- 更新相关实体页面
- 标注新旧知识的矛盾
- 追加操作日志

**结果是**：知识在积累，交叉引用是现成的，矛盾已被标记，综合判断已反映所有已读内容。

详见：
- [[LLM Wiki.md]] — 方法论原文
- [[LLM-WORKFLOW.md]] — 本知识库的配置指南

## 第一步：安装 Obsidian（你的知识管理软件）

### 什么是 Obsidian？

Obsidian 是一个免费的笔记软件。你把笔记存成普通的 Markdown 文本文件，
可以永久保存、自由迁移。和 Notion 不同，你的数据完全在你自己电脑上。

### 下载与安装

#### macOS
1. 打开 https://obsidian.md/download
2. 点击 "Download for macOS"，下载 .dmg 文件
3. 双击下载的 .dmg，把 Obsidian 图标拖到 Applications 文件夹
4. 在启动台找到 Obsidian 图标，双击打开
5. **如果提示"无法验证开发者"**：
   - 打开「系统设置」→「隐私与安全性」
   - 在页面下方找到「仍要打开 Obsidian」按钮，点击
   - 输入密码确认

#### Windows
1. 打开 https://obsidian.md/download
2. 点击 "Download for Windows"，下载 .exe 安装文件
3. 双击运行安装程序，一路点「下一步」即可
4. 桌面会出现 Obsidian 图标，双击启动

#### Linux
1. 打开 https://obsidian.md/download
2. 下载 AppImage 文件
3. 右键点击文件 → 属性 → 权限 → 勾选「允许作为程序执行」
4. 双击运行

#### iPhone / iPad
1. 打开 App Store
2. 搜索 "Obsidian"
3. 点击「获取」安装
4. 完全免费，没有内购

#### Android
1. 打开 Google Play 商店
2. 搜索 "Obsidian"
3. 点击安装
4. 如果无法访问 Google Play：从 https://obsidian.md/download 下载 APK 文件直接安装

⚠️ Obsidian 安装后先**不要**创建 Vault（知识库仓库）。

> Andrej Karpathy（前特斯拉 AI 总监、前 OpenAI 创始成员）

加载 Karpathy 技能后，Claude 会：
- 更懂如何结构化地维护知识库

后续和 Claude 对话时，输入 `/karpathy-guidelines` 可以激活这个技能模式。

关于"行政处罚"和"行政强制措施"的区别，我总是搞混。

## 常见问题 FAQ

### Q1: 这个知识库要钱吗？
**A:** 知识库完全免费（CC BY-SA 4.0 开源协议）。
Obsidian 个人使用免费。
Claude API 按使用量付费，首次赠 $5 额度，一个月备考约花 $2-5。

### Q2: 我需要编程基础吗？
**A:** 不需要。这篇指南就是给零基础的人写的，跟着一步一步做就行。
如果有 Git 基础你会觉得更方便（可以一键更新知识库），但不是必需的。

### Q3: Claude 的回答准确吗？
**A:** 这个知识库的内容经过人工整理和实际考试验证，比直接问通用 AI 准确得多。
但 AI 仍可能犯错，重要知识点请以教材为准。

### Q4: 我的学习数据会不会泄露？
**A:** 不会。API Key 只存在你的电脑上。
你向 Claude 提问时，只有当前问题涉及的文件内容会被发送，你全部的笔记不会上传。

#!/usr/bin/env python3
"""Convert Obsidian wiki-links to standard Markdown links for MkDocs."""

import re
import os
from pathlib import Path

VAULT_ROOT = Path(__file__).parent.parent
DIST_DIR = VAULT_ROOT / "docs" / "content"

# Directories to include in conversion
INCLUDE_DIRS = [
    "wiki",
    "辅导班题目",
    "练题班题库",
    "错题集",
    "时政",
    "陕西省",
    "考前必背-SVG",
    "raw",
    "政策解读",
]


def convert_wikilink(match: re.Match, current_file: Path) -> str:
    """Convert a single [[wikilink]] to standard [text](path) format."""
    inner = match.group(1)
    if "|" in inner:
        target, display = inner.split("|", 1)
    else:
        target = inner
        # Generate display from target (strip extension, use basename)
        display = Path(target).stem

    # Handle external links (URLs)
    if target.startswith("http"):
        return f"[{display}]({target})"

    # Remove .md extension for MkDocs compatibility
    if target.endswith(".md"):
        target = target[:-3]

    # Handle section anchors
    if "#" in target:
        target_path, anchor = target.rsplit("#", 1)
        if target_path.endswith(".md"):
            target_path = target_path[:-3]
        target = f"{target_path}#{anchor}"

    # Determine relative path from current file's position in dist
    # If target is in Vault root, link relative to the converted file's location

    # Check if target file exists in Vault
    target_md = VAULT_ROOT / f"{target}.md"
    target_file = VAULT_ROOT / f"{target}.md" if not target.endswith(".html") else VAULT_ROOT / target

    # For MkDocs, all content is under docs/content/
    # We need relative paths from the current file's output location

    # Simplify: target is relative to Vault root
    rel_target = Path(target)

    if rel_target.suffix:
        pass  # has extension
    else:
        # MkDocs expects .md files without extension in links
        pass

    return f"[{display}]({target})"


def convert_file(src_path: Path, dist_path: Path) -> None:
    """Convert a single Markdown file: wiki-links → standard links."""
    content = src_path.read_text(encoding="utf-8")

    # Pattern for [[link]] or [[link|display]]
    wikilink_pattern = re.compile(r"\[\[([^\]]+)\]\]")

    def replacer(match):
        return convert_wikilink(match, dist_path)

    converted = wikilink_pattern.sub(replacer, content)

    # Also handle ![[image]] embeds
    embed_pattern = re.compile(r"!\[\[([^\]]+)\]\]")
    converted = embed_pattern.sub(r"![\1](\1)", converted)

    dist_path.parent.mkdir(parents=True, exist_ok=True)
    dist_path.write_text(converted, encoding="utf-8")


def main():
    """Convert all Markdown files in INCLUDE_DIRS to DIST_DIR."""
    print("Converting wiki-links to standard Markdown links...")

    count = 0
    for dir_name in INCLUDE_DIRS:
        src_dir = VAULT_ROOT / dir_name
        if not src_dir.exists():
            continue

        for md_file in src_dir.rglob("*.md"):
            rel_path = md_file.relative_to(VAULT_ROOT)
            dist_path = DIST_DIR / rel_path

            convert_file(md_file, dist_path)
            count += 1
            print(f"  ✓ {rel_path}")

    # Also convert root-level .md files
    for md_file in VAULT_ROOT.glob("*.md"):
        rel_path = md_file.relative_to(VAULT_ROOT)
        dist_path = DIST_DIR / rel_path
        convert_file(md_file, dist_path)
        count += 1
        print(f"  ✓ {rel_path}")

    print(f"Done! Converted {count} files.")


if __name__ == "__main__":
    main()
```
