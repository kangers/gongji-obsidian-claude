## 项目概述

公共基础知识考试备考知识库（Obsidian 知识库）。围绕"公共基础知识"（公基）科目，包含备考资料和真题生成工作流。

## 仓库结构

```
考公/
├── 政策解读/              ← 当前工作目录，存放政策文件（PDF）
├── 公基资料库/            ← 公共基础知识全套资料
│   ├── 笔记整理/
│   ├── 公基大默写/
│   ├── 练习题+解析/       ← 按科目分组的练习题
│   ├── 公基必刷1600题/
│   └── 各种 PDF 教材       ← 【01】公基上/下.pdf 等
├── 真题/                   ← 模拟卷（Markdown 格式）
├── 退役军人事务局-培训课/  ← 培训笔记及讲义 PDF
├── OCR文本提取/           ← OCR 输出结果（.txt）
├── AI辅助/                ← AI 生成题目相关
├── ocr_batch.sh           ← 多步 OCR（PDF→图片→tesseract，300 DPI，chi_sim）
├── ocr_fast.sh            ← 快速 OCR（150 DPI，跳过中间步骤）
├── convert_pdfs.sh        ← 简单 pdftotext 批量转换
└── 未命名.md              ← 50套真题生成规范与 JSON 格式定义
```

### 全域目录结构（与政策解读同级）

- **公基资料库/** — 公共基础知识：[01]公基上/下（核心教材）、学霸笔记、行政法、陕西省政府工作报告、练习题+解析、公基必刷1600题
- **真题/** — 模拟卷（Markdown），含 JSON 格式的单选/多选/判断题

- **OCR文本提取/** — OCR 脚本输出的纯文本文件目录
- **AI辅助/** — AI 辅助备考相关资料

## 考试规格（真题生成）

50套完整模拟题，每套 140 题/100 分：

| 题型 | 题数 | 分值 |
|------|------|------|
| 单选题 (single) | 100 题 | 0.5分/题 |
| 多选题 (multiple) | 20 题 | 全对1.5分/漏选0.75分 |
| 判断题 (judge) | 20 题 | 1.0分/题 |

科目权重：政治32% > 法律27% > 管理17% > 经济12% > 公文写作12%

JSON 格式（题目、选项、答案、解析、难度、标签）详见 `未命名.md`。
