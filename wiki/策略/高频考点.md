## 二、法律（权重25%-30%）

### ★★★ 最高频必考

| 考点 | 考点数 | 说明 |
|------|--------|------|
| **宪法修正案（2018）** | 211中核心 | 四个要点：习近平思想、主席任期、设区市立法权、监察委 |
| **★ 退役军人保障法+安置条例** | — | **军转必考**，每套≥5题 |
| **行政法六大基本原则** | 190中核心 | 合法行政（首要）、合理行政、程序正当等 |
| **行政处罚种类与程序** | 同上 | 简易/一般/听证程序的数字条件 |
| **民法典亮点** | 180中核心 | 人格权独立成编、绿色原则、居住权等 |

### ★★ 高频

| 领域 | 考点 | 记忆关键 |
|------|------|----------|
| **宪法** | 国家机构职权表 | 人大(修改宪法) vs 常委会(解释宪法) |
| **宪法** | 公民基本权利 | 选举权=最基本的政治权利，人身自由=最基本的权利 |
| **刑法** | 刑事责任年龄 | 14-16八种罪，16+全责 |
| **刑法** | 正当防卫 vs 紧急避险 | 前者针对不法侵害，后者不得已才可 |
| **刑法** | 刑罚种类 | 主刑(管制/拘役/有期/无期/死刑) vs 附加刑 |
| **行政法** | 行政许可/处罚/强制/复议/诉讼 | 五大制度对比记忆 |
| **民法** | 宣告失踪(2年) vs 宣告死亡(4年) | 年份和公告期易混 |
| **民法** | 诉讼时效 | 一般3年，最长20年 |
| **劳动法** | 加班工资比例 | 150%/200%/300% |

## 三、管理（权重15%-20%）

### ★★ 高频

| 考点 | 说明 |
|------|------|
| 管理四大职能 | 计划(首要)、组织、领导、控制 |
| 管理核心 | 决策 |
| 管理两重性 | 自然属性+社会属性 |
| 马斯洛需求五层次 | 生理→安全→社交→尊重→自我实现 |
| 双因素理论 | 激励因素(工作本身) vs 保健因素(工作环境) |
| 管理幅度与管理层次 | 规模一定时成**反比** |
| 管理者技能 | 高层需要概念技能，基层需要技术技能 |

### ★ 常考

- 泰勒（科学管理之父）、法约尔（一般管理）、韦伯（科层制）、梅奥（霍桑实验）
- 五种管理心理学效应：晕轮、首因、近因、木桶、破窗

## 四、经济（权重10%-15%）

### ★★ 高频

| 考点 | 说明 |
|------|------|
| "看不见的手"(斯密) vs "看得见的手"(凯恩斯) | 市场 vs 政府 |
