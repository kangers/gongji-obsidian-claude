# 行政法与行政诉讼法

> ★ 公共基础知识高频考点。
> 学霸笔记考点数：**190个**

---

## 行政法概述

### 行政法的特征
- 调整行政关系，**规范和控制行政权**（本质）
- 没有统一、完整的法典
- 实体性规范与程序性规范相互交织

### 行政法律关系
- 当事人中必有一方是**行政主体**
- 当事双方地位**不对等**

---

## 行政法基本原则（六大原则）

| 原则 | 内容 |
|------|------|
| **合法行政**（首要） | 法律优先（法已规定不可违）+ 法律保留（法无规定不可为） |
| **合理行政** | 公正平等对待当事人、考虑相关因素、比例原则（合目的性/适当性/损害最小）|
| **程序正当** | 程序公开、公众参与、回避 |
| **高效便民** | 行政效率、便利当事人 |
| **诚实守信** | 行政信息真实、保护信赖利益 |
| **权责统一** | 行政效能、行政责任 |

## 行政主体

### 行政主体资格（"三有"）
- 一有**"权"**（行政权力）
- 二有**"名"**（以自己名义独立从事行政管理）
- 三有**"责"**（独立承担行政责任）

### 分类
- **职权性行政主体** — 成立即获得资格，只能是国家行政机关
- **授权性行政主体** — 权力来自法律、法规、规章的授予

### 派出机关 vs 派出机构
- 派出机关（职权性）：行政公署、区公所、街道办事处
- 派出机构（授权性）：派出所（警告/500元以下罚款）、工商所、税务所

## 行政许可法

> 核心法规：《中华人民共和国行政许可法》（2004.7.1施行）

### 行政许可的特征
- 是一种**依申请**的行政行为（不申请不理）
- 是一种**授益性**行政行为（赋予权利或资格）
- 存在的前提是法律**一般禁止**
- 是一种**要式**行政行为（书面形式）
- 是一种**外部**行政行为

### 行政许可的设定权限（★ 重点辨析）

| 规范层级 | 可否设定许可 | 条件 |
|------|:--:|------|
| **法律** | ✅ 可以 | 无限制 |
| **行政法规** | ✅ 可以 | 法律未设定时 |
| **地方性法规** | ✅ 可以 | 法律、行政法规未设定时 |
| **省级政府规章** | ⚠️ 临时性 | 省级政府规章可设临时许可（**1年**），期满需制定地方性法规 |
| **部门规章** | ❌ 不得设定 | — |
| **其他规范性文件** | ❌ 一律不得设定 | — |

### 行政许可的实施程序
- 申请 → 受理 → 审查 → **决定**（自受理起**20日**内）
- 延长期限最多**10日**
- 联合办理：**45日**内，延长最多**15日**

### 行政许可的听证
- **依职权听证**：涉及公共利益重大事项 → 应当向社会公告
- **依申请听证**：涉及申请人与他人之间重大利益关系 → 告知听证权利
- 听证后根据**听证笔录**作出许可决定

## 行政处罚法

> 核心法规：《中华人民共和国行政处罚法》（2021修订，2021.7.15施行）

### 行政处罚的种类（★ 新版7种）

| 类型 | 种类 | 说明 |
|------|------|------|
| 声誉罚 | **警告、通报批评** | 通报批评为2021新增 |
| 财产罚 | **罚款；没收违法所得、没收非法财物** | |
| 资格罚 | **暂扣许可证件；降低资质等级；吊销许可证件** | "降低资质等级"为2021新增 |
| 行为罚 | **限制开展生产经营活动；责令停产停业；责令关闭；限制从业** | 2021大幅扩充 |
| 人身罚 | **行政拘留** | ★ 15日以下，**只能由法律设定** |

> **不与公务员处分混淆**：记过/记大过/降级/撤职是**公务员处分**，不属于行政处罚！

### 行政处罚的设定权限（★ 重点辨析）

| 规范层级 | 可设定的处罚种类 |
|------|------|
| **法律** | 所有种类，**限制人身自由只能由法律设定** |
| **行政法规** | 除限制人身自由外的所有种类 |
| **地方性法规** | 除限制人身自由、吊销企业营业执照外的处罚 |
| **部门规章** | 仅可设定**警告、通报批评**及一定数额罚款 |
| **地方政府规章** | 仅可设定**警告、通报批评**及一定数额罚款 |

### 行政处罚程序

| 程序 | 适用条件 | 说明 |
|------|------|------|
| **简易程序** | 公民≤**200元**罚款 / 法人≤**3000元**罚款 / 警告 | 可当场处罚 |
| **普通程序** | 不适用简易程序的 | 调查→告知→决定→送达 |
| **听证程序** | 责令停产停业、吊销许可证、较大数额罚款等 | 当事人不承担听证费用 |

### ★ 新《行政处罚法》2021修订亮点
- 增加行政处罚的**定义**
- 完善"**首违不罚**"制度（初次违法且危害轻微并及时改正 → 可不予处罚）
- 延长重点领域**追究时效**至**5年**（一般时效仍为2年）
- 新增行政处罚种类：通报批评、降低资质等级、限制从业等
- 行政处罚权可向**乡镇/街道**下放

## 行政强制法

> 核心法规：《中华人民共和国行政强制法》（2012.1.1施行）

### ★ 行政强制措施 vs 行政强制执行（最常混考点）

| 对比维度 | 行政强制措施 | 行政强制执行 |
|------|------|------|
| **目的** | 预防、制止危害 | 实现已生效的义务 |
| **性质** | **临时性**、中间行为 | **终局性**、最终行为 |
| **前提** | 紧急情况下先行控制 | 当事人**不履行**行政决定 |
| **种类** | ①限制人身自由 ②查封场所设施财物 ③扣押财物 ④冻结存款汇款 | ①加处罚款/滞纳金 ②划拨存款汇款 ③拍卖查封扣押物 ④排除妨碍/恢复原状 ⑤代履行 |

> **口诀**：**措施=封扣押冻**（控制） vs **执行=加划拍代**（实现）

### 行政强制的设定权限

| 强制类型 | 法律 | 行政法规 | 地方性法规 |
|------|:--:|:--:|:--:|
| 行政强制措施 | ✅ | ⚠️ 有限（查封+扣押） | ⚠️ 有限（查封+扣押） |
| 行政强制执行 | ✅ | ❌ | ❌ |

> ★ **行政强制执行只能由法律设定！**

## 行政复议法

> 核心法规：《中华人民共和国行政复议法》（2023修订，2024.1.1施行）

### 行政复议的特征
- 兼具**行政监督**和**行政救济**双重属性
- 审查**合法性+合理性**（行政诉讼大多只审查合法性）
- 原则上采取**书面审查**方式
- **不收费**

### 行政复议的申请
| 项目 | 内容 |
|------|------|
| 申请期限 | 知道行政行为之日起**60日**内（法律规定超过60日的除外） |
| 申请形式 | 书面或口头 |
| 被申请人 | 作出行政行为的行政机关 |

### 复议机关的确定
| 被申请人 | 复议机关 |
|------|------|
| 政府工作部门 | 本级政府 或 上一级主管部门 |
| 地方各级政府 | 上一级政府 |
| 省部级单位 | **原机关**（对复议决定不服可诉讼或申请国务院裁决） |
| 派出机关 | 设立该机关的地方政府 |

### 行政复议期间不停止执行原则 — 例外情形（★ 多选高频）

复议期间行政行为**原则上不停止执行**，但以下**四种情形**可停止：

1. **被申请人**认为需要停止执行的
2. **行政复议机关**认为需要停止执行的
3. **申请人**申请停止执行，复议机关认为合理 → 决定停止
4. **法律规定**停止执行的（如拘留暂缓执行）

> 注意：不包括"第三人申请"！不包括"申请人单方认为"！

### 行政复议决定
| 决定类型 | 适用条件 |
|------|------|
| **维持** | 事实清楚、证据确凿、适用依据正确、程序合法、内容适当 |
| **撤销/变更/确认违法** | 主要事实不清/证据不足/依据错误/程序违法/越权/滥用职权/不当 |
| **责令重作** | 撤销同时可责令被申请人在一定期限内重作 |

### 复议决定期限
- 受理后**60日**内作出（法律规定的少于60日的除外）
- 延长期限最多**30日**

## 行政诉讼法

> 核心法规：《中华人民共和国行政诉讼法》（2014修订，2017修订）

### 行政诉讼的受案范围
✅ **可诉**：具体行政行为（行政处罚/许可/强制/征收/确权等）、行政协议

❌ **不可诉**：
1. 国家行为（国防/外交）
2. **抽象行政行为**（行政法规/规章/规范性文件）
3. 行政机关的**内部行为**（人事处理/奖惩）
4. 法律规定由行政机关**最终裁决**的行为

### 管辖

| 管辖类型 | 规则 |
|------|------|
| **级别管辖** | 一般由**基层法院**管辖 |
| 中院管辖 | ①国务院部门/县级以上政府为被告 ②海关案件 ③本辖区重大复杂案件 |
| **地域管辖** | 一般以**被告所在地**法院管辖 |
| 特殊地域 | ①经复议的案件：原机关/复议机关所在地均可 ②**不动产**：不动产所在地**专属管辖** ③限制人身自由：被告/原告所在地均可 |

### 行政诉讼的原告与被告

| 项目 | 内容 |
|------|------|
| **原告** | 认为行政行为侵犯其合法权益的公民/法人/其他组织 |
| **被告（一般规则）** | 作出行政行为的机关 |
| 复议维持 | ★ **原机关和复议机关为共同被告**（2014修订后） |
| 复议改变 | 复议机关为被告 |
| 复议不作为 | 可诉原行为（原机关为被告）或诉不作为（复议机关为被告） |
| 委托行政 | **委托机关**为被告 |
| 派出机构 | 授权范围内 → 派出机构为被告；超出授权 → 派出机关为被告 |

### 举证责任（★ 核心考点）

| 证据 | 举证方 |
|------|:--:|
| **行政行为合法性** | ★ **被告（行政机关）** |
| 起诉符合条件 | 原告 |
| 行政不作为（已提出申请） | 原告（或被告依法应主动履职的除外） |
| 行政赔偿/补偿（损害事实） | 原告 |

> ★ **被告负主要举证责任**！且被告在诉讼期间**不得自行向原告和证人收集证据**

### 起诉与审理期限

| 项目 | 期限 |
|------|:--:|
| 起诉期限（经过复议） | 收到复议决定书之日起**15日**内 |
| 起诉期限（直接起诉） | 知道行政行为之日起**6个月**内 |
| 起诉期限（最长保护期） | **5年**（不动产**20年**） |
| 一审审理期限 | **6个月**（可延长） |
| 二审审理期限 | **3个月**（收到上诉状起） |

### 行政诉讼判决类型

| 判决类型 | 适用 |
|------|------|
| **驳回诉讼请求** | 行政行为合法 |
| **撤销判决** | 主要证据不足/法律错误/程序违法/越权/滥用职权 |
| **变更判决** | 行政处罚**明显不当**（不是错误！） |
| **确认违法判决** | 撤销无意义/已不可撤销 |
| **确认无效判决** | 行政行为存在重大且明显违法 |
| **履行判决** | 行政机关不履行法定职责 |

## 高频辨析汇总

| 易混项 | 正确 |
|------|------|
| 行政法首要原则 | **合法行政**（非合理/程序正当） |
| 行政法"帝王原则" | **合法行政**（民法的帝王原则才是诚实信用！） |
| 许可设定 | 法律/行政法规/地方性法规/省级规章 → ✅，部门规章 → ❌ |
| 限制人身自由处罚/强制 | **只能由法律设定** |
| 复议申请期限 | **60日** |
| 复议决定期限 | **60日**（可延30日） |
| 起诉期限（直接） | **6个月** |
| 措施 vs 执行 | 措施=封扣押冻（控制）/ 执行=加划拍代（实现） |
| 复议维持被告 | **原机关+复议机关为共同被告** |
| 举证责任 | ★ **被告**对行政行为合法性举证 |
| 复议停止执行 | 被申请人/复议机关/申请人+复议机关同意/法律规定（四种） |
| 授益 vs 负担 | 许可=授益行为 / 处罚=负担行为 |
