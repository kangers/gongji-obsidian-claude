# ★ 中华人民共和国退役军人保障法

> ★ **军转考试必考**，与《安置条例》合计每套试卷至少5题。
> 2021年1月1日起施行，共 **10章85条**。

---

## 立法概述

- **通过**：2020年11月11日，第十三届全国人大常委会第二十三次会议
- **施行**：2021年1月1日
- **地位**：新中国成立以来第一部专门保障退役军人权益的**基础性法律**
- **立法目的**：加强退役军人保障工作，维护退役军人合法权益，让军人成为全社会尊崇的职业

## 核心考点

### 一、管理体制（总则）

| 层级 | 机构 | 职责 |
|------|------|------|
| **国家** | 国务院退役军人工作主管部门（退役军人事务部） | 主管全国退役军人保障工作 |
| **地方** | 县级以上地方人民政府退役军人工作主管部门 | 负责本行政区域工作 |
| **基层** | 乡镇街道、村社区 | 设立退役军人服务站点 |

### 二、★ 退役安置（五种方式）

| 安置方式 | 适用对象 | 要点 |
|----------|----------|------|
| **逐月领取退役金** | 符合条件的军官和军士 | 最新安置方式，按月领取+自主就业结合 |
| **自主就业** | 退役士兵 | 一次性经济补助，享受就业创业扶持 |
| **安排工作** | 符合条件的退役士兵 | 由安置地人民政府安排岗位 |
| **退休** | 符合退休条件的 | 享受退休待遇 |
| **供养** | 以供养方式安置的 | 按照国家有关规定执行 |

### 三、教育培训

- 退役军人接受学历教育 → 享受**学费减免、助学金**等资助
- 国家鼓励**高职扩招**面向退役军人单列计划
- 免费**职业技能培训**，每人可选择**一次免费**培训

### 四、就业创业

| 扶持类型 | 具体措施 |
|----------|----------|
| **就业促进** | 机关/团体/事业单位招录时同等条件优先；设置一定岗位定向招聘 |
| **创业扶持** | 税收优惠、创业担保贷款、创业孵化基地 |
| **公益岗位** | 对就业困难的退役军人给予兜底帮扶 |
| **企业吸纳** | 吸纳退役军人就业的企业享受税收优惠 |

### 五、抚恤优待

| 领域 | 优待内容 |
|------|----------|
| **医疗** | 优抚医院优先诊疗，享受医疗补助 |
| **住房** | 符合条件的优先配租/配售保障性住房 |
| **交通** | 凭优待证免费乘坐市内公交、地铁（各地政策不同） |
| **养老** | 优先入住公办养老机构 |
| **公共服务** | 凭优待证在公共服务方面享受优先 |

### 六、褒扬激励

- 国家建立退役军人**荣誉激励机制**
- 对突出贡献的退役军人给予**表彰、奖励**
- 评选宣传"**最美退役军人**"等先进典型
- 将退役军人荣誉名录载入**地方志**

### 七、法律责任（易考）

| 行为 | 法律责任 |
|------|----------|
| 拒绝接收退役军人 | 责令改正，对直接负责的主管人员依法处分 |
| 非法扣留退役金/优待证 | 责令退还，依法处分 |
| 打击报复退役军人 | 依法给予处分，构成犯罪追究刑事责任 |
