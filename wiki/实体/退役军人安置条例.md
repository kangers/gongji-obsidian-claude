# ★ 退役军人安置条例（国令第787号）

> ★ **军转考试必考**，与《保障法》合计每套试卷至少5题。
> 2024年9月1日起施行，是2026年考试的重中之重。

---

## 立法概述

| 项目 | 内容 |
|------|------|
| **全称** | 退役军人安置条例 |
| **文号** | **国令第787号** |
| **通过** | 2024年7月1日国务院第37次常务会议 |
| **施行** | **2024年9月1日** |
| **性质** | 行政法规，细化《退役军人保障法》 |
| **旧规** | 同时废止《退役士兵安置条例》（2011年） |
| **新意** | 统一军官和士兵的安置制度，首次系统规定逐月领取退役金 |

## ★ 核心考点

### 一、立法亮点（高频判断题）

| 亮点 | 说明 |
|------|------|
| **统一安置制度** | 首次将军官和军士的安置纳入统一法规框架 |
| **逐月领取退役金制度化** | 将自主择业时期的逐月领取退役金做法上升为法规 |
| **安置质量提升** | 明确安排工作的岗位质量要求 |
| **就业创业扶持** | 细化税收优惠、贷款扶持等配套措施 |

### 二、★ 五种安置方式详解

| 安置方式 | 适用对象 | 要点 |
|----------|----------|------|
| **① 逐月领取退役金** | 担任军官/军士满一定年限 | **新制度核心**：每月领取退役金，同时可自主就业创业 |
| **② 自主就业** | 不符合安排工作条件的 | 领取一次性经济补助，各地标准不同（陕西标准需关注） |
| **③ 安排工作** | 符合条件的退役军士和义务兵 | 由安置地人民政府安排岗位，**机关/事业单位/国企为主** |
| **④ 退休** | 符合退休条件的军官/军士 | 按国家有关规定执行 |
| **⑤ 供养** | 被评定为1-4级残疾的 | 由国家供养终身 |

### 三、逐月领取退役金（★ 新增重点）

| 问题 | 答案 |
|------|------|
| 适用条件 | 担任军官/军士满**一定年限**（具体年限按军队有关规定） |
| 发放标准 | 以退役时的级别、衔级、军龄等综合计算 |
| 领取期限 | 终身领取（就业后继续领取） |
| 与自主就业关系 | 可选择同时创业就业，不影响退役金发放 |
| 调整机制 | 根据经济社会发展水平适时调整 |

### 四、安排工作（★ 常考）

| 适用对象 | 说明 |
|----------|------|
| 军士服现役满**12年**的 | ★ 重点记忆数字"**12年**" |
| 服现役期间**平时二等功/战时三等功**以上 | 立功受奖者 |
| 因战致残被评定为**5级至8级**残疾等级 | 因战伤残 |
| 烈士子女 | 烈士子女优先 |

### 安排工作要求
- 安置地人民政府应当在接收退役军士后**6个月内**完成安排工作
- 工作岗位以**事业单位、国有企业**为主
- 安排工作的退役军士享受**不低于本单位同等条件人员**的待遇

### 五、自主就业补助

- 退役军士和义务兵自主就业 → 领取**一次性经济补助**
- 补助标准由**省、自治区、直辖市人民政府**制定（陕西标准需关注）
- 地方标准不得低于**国家标准**

## 与《退役军人保障法》的关系

| 维度 | 保障法（上位法） | 安置条例（下位法） |
|------|-----------------|-------------------|
| **性质** | 法律（全国人大常委会） | 行政法规（国务院） |
| **通过时间** | 2020年11月 | 2024年7月 |
| **施行时间** | 2021年1月1日 | 2024年9月1日 |
| **作用** | 基础性、原则性规定 | 具体化、可操作的实施细则 |
| **安置方式** | 五种方式框架性规定 | 每种方式的具体条件、标准、程序 |
| **考试比重** | 基础考点 | ★★ 2026年考试**重中之重**（最新法规） |
