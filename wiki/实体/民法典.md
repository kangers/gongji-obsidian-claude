# 中华人民共和国民法典

> 2021年1月1日起施行，被称为"社会生活的百科全书"。
> 学霸笔记考点数：**180个**

---

## 民法典概述

### 调整对象
- 调整**平等主体**的自然人、法人和非法人组织之间的**人身关系和财产关系**

### 六项基本原则
| 原则 | 说明 |
|------|------|
| **平等** | 主体地位平等 |
| **公平** | 合理确定权利义务 |
| **自愿** | ★ **核心原则** |
| **诚信** | ★ **帝王原则**（"秉持诚实，恪守承诺"） |
| **公序良俗** | 公共秩序和善良风俗 |
| **绿色** | 节约资源、保护环境（倡导性原则） |

---

## 民法典结构（7编）

| 编 | 名称 | 核心内容 |
|----|------|----------|
| 一 | **总则编** | 民事主体、民事法律行为、代理、诉讼时效 |
| 二 | **物权编** | 所有权、用益物权、担保物权、居住权★ |
| 三 | **合同编** | 合同的订立、效力、履行、违约责任 |
| 四 | **人格权编** ★ | 独立成编是最大亮点 |
| 五 | **婚姻家庭编** | 结婚、家庭关系、离婚 |
| 六 | **继承编** | 法定继承、遗嘱继承 |
| 七 | **侵权责任编** | 过错责任、无过错责任 |

## 核心制度

### 民事主体
- **自然人**：权利能力始于出生终于死亡
- 胎儿利益保护：涉及遗产继承、接受赠与时视为有权利能力
- **法人**：营利法人、非营利法人、特别法人
- **非法人组织**：个人独资企业、合伙企业等

### 民事法律行为
- 有效条件：意思表示真实 + 行为能力 + 不违法
- **无效**：无行为能力人、虚假意思表示、违反强制性规定
- **可撤销**：重大误解、欺诈、胁迫、显失公平

### 宣告失踪与宣告死亡
|  | 宣告失踪 | 宣告死亡 |
|------|----------|----------|
| 下落不明时间 | 满**2年** | 一般**4年**，意外**2年** |
| 公告期 | **3个月** | 一般**1年**，意外证明不可能生存**3个月** |
| 法律效果 | 设立**财产代管人** | **婚姻消灭**、继承开始 |

### 诉讼时效
- 一般诉讼时效：**3年**
- 起算：自知道或应当知道权利受到损害以及义务人之日起
