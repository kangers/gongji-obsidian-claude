# 中国共产党党章

> 考试权重：政治科目内，单选 1-3 题 / 多选偶尔涉及
> 大纲明确列出的独立考点

---

## 党章概述

- **党章**是党的**根本大法**，是全党必须遵循的**总规矩**
- 现行党章：**党的二十大（2022年10月22日）** 通过的最新修正案
- 党章的内容构成：**总纲** + **条文**（共11章55条）

---

## 党的性质（"一个核心、两个先锋队、三个代表"）

- 中国共产党是中国工人阶级的**先锋队**
- 同时是中国人民和中华民族的**先锋队**
- 是中国特色社会主义事业的**领导核心**
- 代表中国**先进生产力**的发展要求、代表中国**先进文化**的前进方向、代表中国**最广大人民**的根本利益

---

## 党的指导思想

> **注意辨析单选题**："马克思主义中国化的最新成果" = 习近平新时代中国特色社会主义思想

| 地位 | 理论成果 | 确立/写入党章时间 |
|------|----------|:--:|
| 理论基础 | 马克思列宁主义 | — |
| 第一次飞跃 | 毛泽东思想 | 七大（1945） |
| 第二次飞跃 | 邓小平理论 | 十五大（1997） |
| 与时俱进 | "三个代表"重要思想 | 十六大（2002） |
| 与时俱进 | 科学发展观 | 十七大（2007）写入党章，十八大（2012）确立为指导 |
| **新的飞跃** | **习近平新时代中国特色社会主义思想** | **十九大（2017）** |

---

## 党的基本路线（社会主义初级阶段）

> "一个中心、两个基本点"

- **以经济建设为中心**（兴国之要）
- **坚持四项基本原则**（立国之本）— 坚持社会主义道路、坚持人民民主专政、坚持中国共产党的领导、坚持马克思列宁主义毛泽东思想
- **坚持改革开放**（强国之路）
- 目标：为把我国建设成为**富强民主文明和谐美丽**的社会主义现代化强国而奋斗

> 注意：十九大在"富强民主文明和谐"后加了 **"美丽"**

---

## 党的组织原则：民主集中制

民主集中制是党的**根本组织原则**，包含六条基本原则：

| 序号 | 原则 | 核心表述 |
|:--:|------|------|
| 1 | "四个服从" | 党员个人服从组织 → 少数服从多数 → 下级服从上级 → **全党服从中央**（核心） |
| 2 | 选举产生 | 各级领导机关由选举产生 |
| 3 | 最高领导机关 | 党的**全国代表大会**和它所产生的**中央委员会** |
| 4 | 上下级关系 | 下级组织向上级组织请示和报告工作 |
| 5 | 集体领导与个人分工 | 凡属重大问题都要**集体讨论**作出决定 |
| 6 | 反对个人崇拜 | 保证党的领导人的活动处于党和人民的监督之下 |

### "四个服从"详解

| 服从关系 | 说明 |
|------|------|
| 个人服从组织 | 最基本 |
| 少数服从多数 | 最常用 |
| 下级服从上级 | 层级关系 |
| **全党服从中央** | ★ **核心和最高原则** |

## 党员

### 入党条件
- 年满**18岁**的中国工人、农民、人员、知识分子和其他社会阶层的先进分子
- 承认党的**纲领和章程**
- 愿意参加党的一个组织并在其中积极工作、执行党的**决议**
- 按期**交纳党费**

### 入党誓词（十九大/二十大版本）
> 我志愿加入中国共产党，拥护党的纲领，遵守党的章程，履行党员义务，执行党的决定，严守党的纪律，保守党的秘密，对党忠诚，积极工作，为共产主义奋斗终身，随时准备为党和人民牺牲一切，永不叛党。

### 预备党员
- 预备期：**一年**
- 预备党员的义务同正式党员，但**没有表决权、选举权和被选举权**

### 党员义务（八项）
- 最核心义务：**学习**（学习党的理论 + 业务知识）+ **带头**（先锋模范作用）

## 党的组织制度

### 党的各级组织

| 层级 | 任期 |
|------|:--:|
| 党的全国代表大会 | **5年** |
| 党的省/自治区/直辖市代表大会 | 5年 |
| 党的设区的市代表大会 | 5年 |
| 党的县（旗）/不设区的市/市辖区代表大会 | 5年 |
| 党的基层委员会 | 5年 |
| 党的总支部委员会/支部委员会 | **3年** |

### 党的中央组织
- ★ 党的最高领导机关：**全国代表大会**和它所产生的**中央委员会**
- 在全国代表大会闭会期间，**中央委员会**执行全国代表大会的决议，领导党的全部工作
- 中央委员会全体会议选举产生：中央政治局、中央政治局常务委员会、中央委员会总书记
- 中央委员会总书记由**中央委员会全体会议**选举产生

## 党的纪律

### 六大纪律
| 纪律 | 说明 |
|------|------|
| 政治纪律 | ★ 最重要、最根本、最关键的纪律 |
| 组织纪律 | 民主集中制、请示报告 |
| 廉洁纪律 | 廉洁自律 |
| 群众纪律 | 密切联系群众 |
| 工作纪律 | 履行职责 |
| 生活纪律 | 生活作风 |

### 纪律处分种类（5种）
| 处分 | 说明 |
|------|------|
| **警告** | 最轻 |
| **严重警告** | 一年内不得在党内提升职务 |
| **撤销党内职务** | 撤销由党内选举或组织任命的职务 |
| **留党察看** | 最长不超过**2年**，期间无表决权、选举权、被选举权 |
| **开除党籍** | ★ **最高处分**，5年内不得重新入党 |

> 与公务员处分的区分：公务员处分=警告/记过/记大过/降级/撤职/**开除**（6种），党纪处分=上述5种
