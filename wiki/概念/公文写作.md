## 公文种类（15种法定公文）

| 分类 | 文种 | 方向 | 用途 |
|------|------|------|------|
| 指令性 | 决定 | 下行 | 重要事项或重大行动作出安排 |
| 指令性 | 决议 | 下行 | 会议讨论通过的重大决策 |
| 指令性 | 命令（令） | 下行 | 发布法规、任免、嘉奖等 |
| 知照性 | 公告 | 下行 | 向国内外宣布重要事项 |
| 知照性 | 通告 | 下行 | 公布社会各有关方面应遵守事项 |
| 知照性 | 通知 | 下行/平行 | 批转、转发、传达、告知 |
| 知照性 | 通报 | 下行 | 表彰先进、批评错误、传达情况 |
| 知照性 | 公报 | 下行 | 公布重要决定或重大事项 |
| 报请性 | 报告 | 上行 | 汇报工作、反映情况 |
| 报请性 | 请示 | 上行 | 请求指示、批准 |
| 报请性 | 议案 | 平行 | 政府向人大提事项 |
| 商洽性 | 函 | 平行 | 不相隶属机关间商洽工作 |
| 指导性 | 意见 | 通用 | 对重要问题提出见解和处理办法 |
| 规范性 | 批复 | 下行 | 答复下级请示事项 |
| 记载性 | 纪要 | 内部 | 记载会议情况和议定事项 |

### 易混淆文种辨析
- **公告 vs 通告**：公告向国内外宣布，通告面向社会各有关方面
- **请示 vs 报告**：请示需批复，报告不需批复（报告不得夹带请示事项）
- **通知 vs 通报**：通知用于布置工作、告知事项，通报用于表彰批评、传达情况

## 公文格式规范

### 版头部分（上行排列）
- **份号** — 涉密公文标注
- **密级和保密期限** — 绝密、机密、秘密
- **紧急程度** — 特急、加急
- **发文机关标志** — 红色小标宋体字
- **发文字号** — 机关代字 + 年份 + 顺序号（如：国发〔2024〕1号）
- **签发人** — 上行文必须标注

### 主体部分
- **标题** — 发文机关 + 事由 + 文种（三要素）
- **主送机关** — 公文的主要受理机关
- **正文** — 开头 + 主体 + 结尾
- **附件说明** — 附件顺序号和名称
- **发文机关署名**
- **成文日期** — 领导人签发日期
- **印章** — 盖红色公章
- **附注** — 公文印发传达范围等

### 版记部分
- **抄送机关**
- **印发机关和印发日期**
- **页码**

## 行文规则

### 上行文规则
- 原则上**一个主送机关**
- 不得抄送下级机关
- 请示应当**一文一事**
- 报告不得夹带请示事项

### 下行文规则
- 主送受理机关，根据需要抄送相关机关
- 涉及其他部门职权范围的，应协商一致

### 平行文规则
- 不相隶属机关之间行文用**函**
- 同级机关可**联合行文**

## 公文处理

### 收文程序
签收 → 登记 → 初审 → 承办 → 传阅 → 催办 → 答复

### 发文程序
拟稿 → 审核 → 签发 → 登记 → 印制 → 核发

### 注意事项
- 公文用纸幅面：A4型纸
- 装订左侧装订
- 页码用4号半角宋体阿拉伯数字
