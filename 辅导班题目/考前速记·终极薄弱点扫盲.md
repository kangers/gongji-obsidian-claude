## 🔴 补录二：行政法三大板块——行政处罚 vs 行政强制措施 vs 行政强制执行

> 你最薄弱的行政法板块，三者的区分是考试绝对重点

### 一、三兄弟一句话区分

| 类型 | 一句话 | 例子 |
|------|--------|------|
| **行政处罚** | 因为你**违法了**，所以**惩罚你** | 罚款、拘留、吊销执照 |
| **行政强制措施** | 为了防止危险/保全证据，**暂时控制** | 查封、扣押、冻结 |
| **行政强制执行** | 你**不履行义务**，**强制你履行** | 加处罚款、划拨存款、代履行 |

### 二、行政处罚（因为你做错了事）

**定义**：行政机关依法对违反行政管理秩序的公民、法人或其他组织，以**减损权益或增加义务**的方式予以**惩戒**的行为。

#### 行政处罚的种类（四大类，必考）

| 类别 | 种类 | 关键词 |
|------|------|------|
| **申诫罚**（声誉罚） | **警告**、通报批评 | 精神上谴责 |
| **财产罚** | **罚款**、没收违法所得、没收非法财物 | 钱和物 |
| **资格罚**（行为罚/能力罚） | **暂扣**许可证、**吊销**许可证、责令停产停业、限制从业 | 剥夺资格/能力 |
| **人身罚**（自由罚） | **行政拘留** | ⚠️ 只能由**法律**设定！ |

#### 行政处罚的设定权限（必考层级表）

```
法律 ────────── 可以设定所有种类处罚；【独占】限制人身自由（行政拘留）
行政法规 ───── 除限制人身自由以外的所有处罚
地方性法规 ─── 除限制人身自由、吊销企业营业执照以外的处罚
部门规章 ───── 只能设定【警告、通报批评】和【一定数额罚款】
地方政府规章 ── 同上（限额由省级人大常委会定）
```

> ⚠️ 口诀：**法律全都能，行政不能拘，地方两不能（不能拘+不能吊企照），规章只能警和钱**

#### 行政处罚程序关键数字

| 程序 | 时限/条件 |
|------|------|
| **追诉时效** | 一般**2年**；涉及生命健康安全/金融安全且有危害后果=**5年** |
| **一般程序决定期限** | 立案之日起**90日**内 |
| **听证权利** | 作出**责令停产停业、吊销许可证、较大数额罚款**等决定前，应告知当事人有要求听证的权利 |
| **不罚年龄** | 不满**14周岁**→不予处罚 |
| **从轻/减轻年龄** | 已满14不满18→**应当**从轻或减轻 |
| **首违不罚** | 轻微+及时改正+无危害后果→**应当不予**处罚 |
| **无主观过错** | 当事人有证据足以证明→**不予**处罚 |

#### 行政处罚 vs 行政处分

| | 行政处罚 | 行政处分 |
|------|------|------|
| 对象 | 公民/法人/其他组织（**外部**） | 公务员（**内部**） |
| 种类 | 警告/罚款/拘留/吊销等 | 警告/记过/记大过/降级/撤职/开除 |
| 救济 | 行政复议/行政诉讼 | 申诉 |

### 三、行政强制措施（暂时性控制，预防性的）

**定义**：行政机关在行政管理过程中，为**制止违法行为、防止证据损毁、避免危害发生、控制危险扩大**等情形，依法对公民人身自由实施**暂时性限制**，或对财物实施**暂时性控制**的行为。

#### 行政强制措施的种类

| 种类 | 具体方式 |
|------|------|
| 限制人身自由 | 强制带离、强制隔离、强制戒毒、**保护性约束** |
| 查封 | 对**场所/设施/财物**贴封条就地封存 |
| 扣押 | 对**动产**转移到指定地点 |
| 冻结 | 对**存款/汇款**暂停支付 |

#### 核心考点

| 考点 | 答案 |
|------|------|
| 冻结期限 | **30日**（可延长，最长不超60日） |
| 实施时间限制 | **不得在夜间或法定节假日**实施（紧急情况除外） |
| 设定权限 | **法律**可以设定所有行政强制措施；行政法规可以设定除限制人身自由、冻结存款汇款以外的；地方性法规可设定**查封和扣押** |
| 性质 | **暂时性**！不是终局性的 |
| 目的 | 预防/制止/控制，不是惩罚！ |

### 四、行政强制执行（你不主动履行，我来强制）

**定义**：行政机关或行政机关申请法院，对**不履行行政决定**的公民/法人，依法**强制履行义务**的行为。

#### 行政强制执行的方式

| 方式 | 说明 |
|------|------|
| **加处罚款或滞纳金** | ⚠️ 这属于执行罚，不是行政处罚！ |
| 划拨存款、汇款 | 银行直接划走 |
| 拍卖或处理查封扣押财物 | 变现抵偿 |
| 排除妨碍、恢复原状 | 如拆除违建 |
| **代履行** | 行政机关或第三方代替当事人履行，费用由当事人承担 |

#### 核心考点

| 考点 | 答案 |
|------|------|
| 执行主体 | 有强制执行权的行政机关**自行执行**；无权的**申请法院**执行 |
| 催告 | 强制执行前**必须催告**当事人履行 |
| 协议 | 行政机关可在**不损害公共利益和他人合法权益**的前提下，与当事人达成**执行协议** |
| 时间限制 | 不得在**夜间或法定节假日**实施（紧急除外） |
| 方式限制 | 不得对**居民生活采取停止供水、供电、供热、供燃气**等方式迫使履行 |

### 五、三者核心对比表

| 对比维度 | 行政处罚 | 行政强制措施 | 行政强制执行 |
|------|------|------|------|
| **目的** | 惩戒违法 | 预防/制止/控制 | 强制履行义务 |
| **前提** | 当事人**违法** | 存在危险/需要取证 | 当事人**不履行义务** |
| **性质** | 终局性（惩罚结束就完了） | **暂时性** | 执行完毕就完了 |
| **典型例子** | 罚款、拘留、吊销执照 | 查封、扣押、冻结 | 加处罚款、代履行、划拨存款 |
| **是否有惩罚性** | ✅ 有 | ❌ 无（控制性） | ❌ 无（执行性） |

### 六、最容易搞混的

| 易混点 | 正确答案 |
|------|------|
| **加处罚款**属于什么？ | 行政**强制执行**（执行罚），不是行政处罚！ |
| **责令停产停业**属于什么？ | 行政**处罚**（资格罚） |
| **查封**属于什么？ | 行政**强制措施** |
| **吊销许可证**属于什么？ | 行政**处罚** |
| **暂扣许可证**属于什么？ | 行政**处罚**（资格罚） |
| 罚款 vs 加处罚款 | 罚款=行政处罚；加处罚款=行政强制执行 |

### 二、行政许可法——核心考点

| 考点 | 答案 |
|------|------|
| 行政许可定义 | 行政机关根据申请，经审查，准予从事**特定活动**的行为 |
| 许可决定期限 | 受理后**20日内**（可延长10日） |
| 材料补正期限 | 当场或**5日内**一次性告知 |
| 听证申请 | 申请人/利害关系人被告知后**5日内**提出；行政机关**20日内**组织 |
| 注销 vs 撤销 vs 吊销 | 注销=到期/终止；撤销=当初不该批；吊销=处罚（违法） |
| 信赖保护 | 变更/撤回许可→造成损失→应当**补偿**（非赔偿） |
| 许可不得转让原则 | 除法律、法规规定可转让的，**不得擅自转让** |
| 许可评价 | 设定机关应定期评价，实施机关可适时评价 |

### 三、监察法——核心考点

| 考点 | 答案 |
|------|------|
| 监察委性质 | **政治机关**（非行政机关、非司法机关） |
| 谁产生监察委 | **同级人大** |
| 监察委对谁负责 | 产生它的**国家权力机关**+**上一级监察委** |
| 监察对象（六类人） | 公务员、参公人员、授权委托从事公务人员、国企管理人员、公办单位管理人员、基层群众自治组织管理人员 |
| 主要职责 | 监督、调查、**处置** |
| 留置期限 | 一般**3个月**（可延长一次，最长6个月） |
| 监察委与纪委关系 | **合署办公**（一套人马两块牌子） |
| 独立行使监察权 | 不受行政机关、社会团体和个人的干涉 |
| 监察权不能委托 | 监察权只能由监察机关自己行使，不得委托 |

### 四、公务员法——核心考点

| 考点 | 答案 |
|------|------|
| 公务员定义 | 依法履行公职、纳入国家行政编制、由国家财政负担工资福利 |
| 公务员领导职务层级 | 国家级/省部级/厅局级/县处级/乡科级（正副共10级） |
| 处分种类 | 警告(6月)/记过(**12**月)/记大过(18月)/降级(24月)/撤职(24月)/开除 |
| 处分期不得晋升 | 处分期内**一律不得晋升职务、职级和级别** |
| 处分期工资 | 警告→工资不变；记过/记大过→不升工资档次；降级/撤职→降工资 |
| 退休后从业限制 | 原系领导成员：**3年内**不得从事与原工作直接相关的营利性活动；其他公务员：**2年内** |
| 辞职后从业限制 | 同退休后 |
| 不得兼职 | 不得在**营利性组织**中兼职 |
| 回避制度 | 夫妻/直系/三代旁系/近姻亲→不得在同一机关担任直接上下级/监察/人财物 |
| 地域回避 | 乡级/县级主要领导：不得在**本人成长地**任职 |
| 定期考核 | 优秀/称职/基本称职/**不称职** |
