# 毛泽东著作考点全梳理

> 按时间线排列，标注发展阶段、核心内容和常考单选题/判断题

---

## 一、毛泽东思想发展四阶段

| 阶段 | 时间 | 标志性事件/著作 | 常考点 |
|------|------|----------------|--------|
| **萌芽** | 1921-1927（建党~大革命） | 《中国社会各阶级的分析》《湖南农民运动考察报告》 | 提出**新民主主义革命基本思想** |
| **初步形成** | 1927-1935（土地革命前期） | 《星星之火，可以燎原》《反对本本主义》 | **"农村包围城市、武装夺取政权"理论** = 初步形成标志 |
| **走向成熟** | 1935-1945（土地革命后期~抗战） | 《实践论》《矛盾论》《论持久战》《新民主主义论》 | 七大确立为指导思想，**新民主主义理论完整阐述**=完全成熟 |
| **继续发展** | 1945-1976（解放战争~建国后） | 《论人民民主专政》《论十大关系》《关于正确处理人民内部矛盾的问题》 | 社会主义革命和建设理论 |

## 二、按时间排序——核心著作考点

### 📖 1. 《中国社会各阶级的分析》（1925年）

| 考点 | 内容 |
|------|------|
| 时期 | **萌芽时期**（建党初期） |
| 核心贡献 | 分清敌友是革命首要问题；指出**无产阶级的最广大和最忠实的同盟军是农民** |
| 易错 | 不是"初步形成"时期著作！萌芽期代表作 |

---

### 📖 2. 《湖南农民运动考察报告》（1927年）

| 考点 | 内容 |
|------|------|
| 时期 | 萌芽时期 |
| 核心贡献 | 充分肯定农民在革命中的作用；提出**贫农是革命先锋** |

---

### 📖 3. 《中国的红色政权为什么能够存在？》（1928年）

| 考点 | 内容 |
|------|------|
| 时期 | **初步形成**时期 |
| 核心贡献 | 提出"**工农武装割据**"思想 |

---

### 📖 4. 《井冈山的斗争》（1928年）

| 考点 | 内容 |
|------|------|
| 时期 | 初步形成时期 |
| 核心贡献 | 进一步阐述工农武装割据 |

---

### 📖 5. 《星星之火，可以燎原》（1930年）

| 考点 | 内容 |
|------|------|
| 时期 | **初步形成**时期 |
| 核心贡献 | 实际上确立了"**农村包围城市、武装夺取政权**"的革命道路 |
| 标志意义 | 🔥 "农村包围城市"理论的提出 = **毛泽东思想初步形成的标志** |
| 易错 | 是"初步形成"标志，不是"成熟"标志！ |

---

### 📖 6. 《反对本本主义》（1930年）

| 考点 | 内容 |
|------|------|
| 时期 | 初步形成时期 |
| 核心贡献 | 提出"**没有调查，就没有发言权**"；反对教条主义 |
| 地位 | 毛泽东第一篇反对教条主义的重要文献 |

---

### 📖 7. 《实践论》（1937年）

| 考点 | 内容 |
|------|------|
| 时期 | **延安时期** / 走向成熟 |
| 核心内容 | 认识论——实践是认识的基础和检验真理的标准 |
| 易错 | ⚠️ 写于延安（1937），不是井冈山时期！ |

---

### 📖 8. 《矛盾论》（1937年）

| 考点 | 内容 |
|------|------|
| 时期 | 延安时期 / 走向成熟 |
| 核心内容 | 辩证法——对立统一规律是唯物辩证法的根本规律；矛盾普遍性与特殊性 |
| 易错 | 同上，延安时期。和《实践论》合称"两论" |

---

### 📖 9. 《论持久战》（1938年）

| 考点 | 内容 |
|------|------|
| 时期 | **延安时期** / 抗战时期 |
| 核心内容 | 系统阐明**抗日持久战**战略总方针 |
| 核心观点 | 抗日战争三阶段：**战略防御→战略相持→战略反攻** |
| 地位 | 🔥 中国共产党领导抗日战争的**纲领性文献** |
| 易错 | 写于**1938年延安**，属延安时期著作（模拟11/12反复考） |

---

### 📖 10. 《<共产党人>发刊词》（1939年）

| 考点 | 内容 |
|------|------|
| 时期 | 延安时期 |
| 核心贡献 | 🔥 提出**三大法宝**：**统一战线、武装斗争、党的建设** |
| 易错 | "三大法宝"出自这里！容易和"三大作风"搞混 |

---

### 📖 11. 《中国革命和中国共产党》（1939年）

| 考点 | 内容 |
|------|------|
| 时期 | 延安时期 |
| 核心贡献 | 🔥 **首次提出"新民主主义"这一科学概念** |
| 易错 | 首次**提出**概念=此书；系统**阐述**=《新民主主义论》（模拟11错过） |

---

### 📖 12. 《新民主主义论》（1940年）

| 考点 | 内容 |
|------|------|
| 时期 | **延安时期** / 走向成熟 |
| 核心贡献 | 🔥 **系统阐述**新民主主义革命理论 |
| 标志意义 | 新民主主义理论完整形成 = **毛泽东思想成熟的主要标志** |
| 易错 | "首次提出"概念 ≠ 本书！首次提出在《中国革命和中国共产党》 |

---

### 📖 13. 《论联合政府》（1945年，七大政治报告）

| 考点 | 内容 |
|------|------|
| 时期 | 延安时期 |
| 核心贡献 | 🔥 提出**三大优良作风**：理论联系实际、密切联系群众、批评与自我批评 |
| 易错 | "三大作风"出自这里！容易和"三大法宝"（出自发刊词）搞混 |

---

### 📖 14. 《在晋绥干部会议上的讲话》（1948年）

| 考点 | 内容 |
|------|------|
| 时期 | 解放战争时期 |
| 核心贡献 | 🔥 完整提出**新民主主义革命总路线** |
| 总路线 | 无产阶级领导的，人民大众的，反对**帝国主义、封建主义、官僚资本主义**的革命 |
| 易错 | 官僚资本主义是三大革命对象之一，容易漏 |

---

### 📖 15. 《论人民民主专政》（1949年）

| 考点 | 内容 |
|------|------|
| 时期 | 建国前夕 |
| 核心贡献 | 阐述新中国国家性质——**人民民主专政**（对人民民主+对敌人专政） |

---

### 📖 16. 《论十大关系》（1956年4月）

| 考点 | 内容 |
|------|------|
| 时期 | 社会主义建设初期 |
| 核心贡献 | 🔥 标志着中国共产党**开始探索适合中国国情的社会主义建设道路** |
| 基本方针 | **调动一切积极因素为社会主义建设服务** |
| 易错 | 是"开始探索"的标志，不是"找到"！ |

---

### 📖 17. 《关于正确处理人民内部矛盾的问题》（1957年）

| 考点 | 内容 |
|------|------|
| 时期 | 社会主义建设初期 |
| 核心贡献 | 区分两类矛盾：**敌我矛盾**（对抗性→专政）vs **人民内部矛盾**（非对抗性→民主方式解决） |
| 工业化道路 | 🔥 中国工业化道路=**重工业、轻工业和农业的发展关系问题** |
| 易错 | 人民内部矛盾用**民主**解决，不是专政！ |

## 三、高频对比辨析

### 🔥 三大法宝 vs 三大作风 vs 三大灵魂

| 概念 | 内容 | 出处 |
|------|------|------|
| **三大法宝** | 统一战线、武装斗争、党的建设 | 《<共产党人>发刊词》(1939) |
| **三大优良作风** | 理论联系实际、密切联系群众、批评与自我批评 | 《论联合政府》(1945) |
| **毛泽东思想活的灵魂** | 实事求是、群众路线、独立自主 | 十一届六中全会《历史决议》 |

### 🔥 "首次提出" vs "系统阐述"

| 概念 | 首次提出 | 系统阐述 |
|------|---------|---------|
| 新民主主义 | 《中国革命和中国共产党》(1939) | 《新民主主义论》(1940) |
| 农村包围城市 | 《星星之火，可以燎原》(1930) | 后续一系列著作 |

### 🔥 萌芽 vs 初步形成 vs 成熟

| 阶段 | 代表著作 | 标志 |
|------|---------|------|
| 萌芽 | 《中国社会各阶级的分析》《湖南农民运动考察报告》 | 提出基本思想 |
| 初步形成 | 《星星之火，可以燎原》《反对本本主义》 | 农村包围城市道路理论 |
| 成熟 | 《新民主主义论》《<共产党人>发刊词》《论联合政府》 | 新民主主义理论完整形成 |

### 🔥 重要会议上的著作/讲话

| 会议 | 著作/内容 | 核心 |
|------|----------|------|
| 八七会议(1927) | 提出"**枪杆子里出政权**" | 武装斗争 |
| 古田会议(1929) | 《关于纠正党内的错误思想》 | 思想建党、政治建军 |
| 遵义会议(1935) | 独立自主解决中国革命问题 | 生死攸关的转折点 |
| 七大(1945) | 《论联合政府》+确立毛泽东思想 | 指导思想 |

## 四、常考判断题速记

| 判断 | 对错 | 理由 |
|------|:--:|------|
| 《实践论》《矛盾论》写于井冈山时期 | ❌ | **延安时期**(1937) |
| "农村包围城市"标志着毛泽东思想成熟 | ❌ | 标志着**初步形成** |
| 新民主主义理论完整形成标志着毛泽东思想成熟 | ✅ | |
| 《新民主主义论》首次提出"新民主主义"概念 | ❌ | 首次提出在《中国革命和中国共产党》 |
| 三大法宝=理论联系实际、密切联系群众、批评自我批评 | ❌ | 那是**三大作风**！三大法宝=统战+武装+党建 |
| 人民内部矛盾用专政方式解决 | ❌ | 用**民主**方式！敌我矛盾才用专政 |
| 《论十大关系》标志着找到社会主义建设道路 | ❌ | 标志着**开始探索**，不是找到 |
| 毛泽东思想=毛泽东个人的思想 | ❌ | 是**集体智慧**的结晶 |
| 毛泽东思想在七大被确立为指导思想 | ✅ | 1945年七大 |
| 毛泽东思想是第一次理论飞跃的成果 | ✅ | 第二次飞跃=中国特色社会主义理论体系 |
