## 部分题目解析

### 马哲
**4.** 矛盾两种基本属性 = 同一性（统一） + 斗争性（对立）。普遍性和特殊性是"矛盾问题的精髓"，不是基本属性。

**9.** B"存在即被感知"= 贝克莱，典型主观唯心。A"理在事先"= 朱熹客观唯心，C"绝对精神"= 黑格尔客观唯心，D"道生万物"= 老子客观唯心。

**10.** 三大规律分工：对立统一→源泉动力，质量互变→过程状态，否定之否定→道路趋势。

### 党史
**11.** 毛泽东思想活的灵魂 = 实事求是 + 群众路线 + 独立自主，**不含**武装斗争。

**13.** 区别于其他政党的显著标志 = **三大作风**（理论联系实际、密切联系群众、批评与自我批评）。

**15.** 三大法宝 = 统一战线 + 武装斗争 + 党的建设。注意"党的领导"不是，是"党的建设"。

### 中特/习思想
**20.** 三个代表→"建设什么样的党、怎样建设党"；邓小平→"什么是社会主义、怎样建设社会主义"；科学发展观→"实现什么样的发展、怎样发展"。

**27.** 第一个答案（毛泽东）= 民主；第二个答案（习近平）= 自我革命。

**28.** 总体国家安全观：以**人民安全**为宗旨，以政治安全为根本，以经济安全为基础。

**30.** "中国共产党领导是中国特色社会主义最本质的特征"在**十九大**明确提出，2018年写入宪法。

### 党章
**32-33.** 宗旨 = 为人民服务；最高理想 = 共产主义（不是"民族复兴"）。

### 时政
**37-38.** 国内出游+16.2%，入境+17.1%。

**39.** 十四五城镇新增就业累计超过**6000万**（每年约1200万×5年）。

**44.** 劳动年龄人口平均受教育年限 → **11.3年**。

**45.** 2025年成功举办了**第九届亚冬会+第十五届全运会**。十四运是2021年在陕西办的。

### 法律
**53.** 宪法修正：常委会或**1/5**代表提议，**2/3**以上多数通过。

**54.** D劳动权是社会经济权利，不是政治权利。

**55.** 劳动权和受教育权 → 既是权利又是义务。

**57.** D暂扣许可证 = 资格罚（行为罚）。A警告=申诫罚，B罚款=财产罚，C行政拘留=人身罚。

**62.** 公务员处分期：警告6月，记过**12月**，记大过18月，降级/撤职24月。

**63.** 行政许可材料补正 → **5日**内一次性告知。

**70.** "自动退休"不是正式安置方式。安置方式：调动、经济补偿金、复员、**退休**（符合条件的）。

### 经济
**77.** 互补品价格↑ → 主商品需求**减少**（如汽油涨价，汽车需求减少）。

**80.** 支出法GDP = 消费+投资+政府购买+净出口；**中间产品**不计入（避免重复计算）。

### 管理
**82.** 泰罗=科学管理之父；法约尔=现代经营管理之父；韦伯=组织理论之父；梅奥=人际关系学说创始人。

**88.** 公共政策的**核心环节是政策执行**。

**89.** 赫伯特·西蒙 → **有限理性决策**。

### 公文
**92.** D发文字号属于**版头**部分，不在主体部分。

**93.** 上行文（请示、报告等）都应当标注签发人姓名。

**97.** 15种法定公文不含**简报**。简报是事务文书。

**120.** 密级 = 绝密、机密、**秘密**。"保密"不是密级。

### 判断详解
**123. B** 七大（1945年）确立毛泽东思想。

**124. B** 八大提出的主要矛盾是"先进工业国vs落后农业国"之间的矛盾；"人民日益增长的物质文化需要同落后的社会生产之间的矛盾"是**十一届六中全会**提出的。

**132. B** 党的最高领导机关 = 全国代表大会 + 中央委员会（不是政治局常委会）。

**134. B** 县级以上政府为被告 → **中级**人民法院管辖。

**141. B** 需求定理：价格↑ → 需求量**减少**（反向关系）。

**142. B** 提高再贴现率 → 货币收缩 → **紧缩性**货币政策。

**148. B** 报告中**绝对不能夹带**请示事项！
